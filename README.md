# mcp-tcmb-exchange

TCMB (Türkiye Cumhuriyet Merkez Bankası) resmi günlük döviz kurlarını MCP (Model Context Protocol) üzerinden sunan projedir.

## Özellikler

- Gerçek zamanlı TCMB kur verisi (USD, EUR, vb.)
- Tamamı `Decimal` bazlı sorunsuz kur çevirimi.
- Çoklu kur çevirimi ve geçmiş tarih desteği.

## Kurulum
```bash
pip install mcp-tcmb-exchange
```
veya uvx ile:
```bash
uvx mcp-tcmb-exchange
```
