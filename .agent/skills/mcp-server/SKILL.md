name: MCP Server Development
description: MCP ile Python kullanarak Server Geliştirme Temelleri
---

Bu proje, bir stdio temelli Server geliştirmek için MCP ve Async/Await Python mantığını kurgulamaktadır.
Client sınıfı HTTP isteklerini yönlendirir iken Server sınıfı `@server.list_tools()` üzerinden dışarıya araç sunar.
